# Exact-Change Format Rules

Always show the exact before/after text and a short reason.

## Required fields per change
- Before — verbatim text from the source, quoted.
- After — verbatim text in the edited version, quoted.
- Reason — one short sentence for the edit.

## Canonical pattern
- [CHANGE] Short description
  - Before: `...`
  - After: `...`
  - Reason: ...

## Rules
- Include at least one lineified change per edit, never paraphrase.
- If the change is an addition, Before: "(not present)".
- If the change is a removal, After: "(removed)".
- Keep reasons concise and neutral.
- Editorial review section must appear at the end of the same document.
