# Minimal-Edit Change List Example

Use this format when user asks for change list only, without full article body.

## Changes
- [CHANGE] Short description of edit.
- [CHANGE] Another edit.

## Editorial Review

### Derogatory or improper terms
- None found. / List with line-level evidence.

### Incorrect or inaccurate statements
- None found. / List with corrections.

### Repeated statements
- None found. / List only exact duplicated body sentences.

### Grammar / style notes
- "quote" — note only, no edit made.
