# Exact Change List Example

Use when the user asks for exact terms changed with reasons in neat bullet format.

## Format

### Changes
- [CHANGE] Short description
  - Before: `exact quoted term or sentence`
  - After: `exact quoted term or sentence`
  - Reason: why the edit was made.

- [CHANGE] Another edit
  - Before: `...`
  - After: `...`
  - Reason: ...

## Editorial review
- Derogatory or improper terms: none found.
- Incorrect or inaccurate statements: none found.
- Repeated statements: none found.
- Grammar / style notes: "..." — note only.
